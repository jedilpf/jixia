## 🎨 制作步骤

### 方法一：使用 Word（推荐）

**步骤**：

1. **打开 Word**
   - 新建空白文档
   - 页面设置：A4 纸
   - 页边距：上下 2.54cm，左右 3.17cm

2. **输入封面内容**
   - 复制上面的模板内容
   - 调整字体和大小：
     - 标题：黑体，二号，居中
     - 软件名称：宋体，三号，加粗
     - 其他文字：宋体，三号
     - 行距：1.5 倍行距

3. **添加边框**
   - 插入 → 形状 → 矩形
   - 绘制页面边框
   - 设置线条颜色：黑色
   - 设置线条粗细：1.5 磅

4. **填写信息**
   - 申请人姓名
   - 联系电话
   - 电子邮箱
   - 申请日期

5. **保存并打印**
   - 保存为：`材料封面.docx`
   - 打印 1 份
   - 使用 A4 纸

---

### 方法二：使用画图工具

**步骤**：

1. **打开画图**
   - Win+R → 输入`mspaint` → 回车

2. **设置画布**
   - 文件 → 属性
   - 宽度：2480 像素
   - 高度：3508 像素
   - 单位：像素
   - 分辨率：300 DPI

3. **绘制边框**
   - 选择矩形工具
   - 设置轮廓粗细：5 像素
   - 颜色：黑色
   - 绘制页面边框

4. **添加文字**
   - 选择文字工具（A）
   - 输入封面内容
   - 调整字体大小：
     - 标题：72 号
     - 软件名称：48 号
     - 其他文字：36 号

5. **保存并打印**
   - 保存为：`材料封面.png`
   - 打印 1 份
   - 使用 A4 纸

## 📝 填写说明

### 申请人信息

**个人申请**：
- 申请人：填写身份证上的姓名
- 联系电话：手机号码
- 电子邮箱：常用邮箱

**企业申请**：
- 申请人：填写企业全称（与营业执照一致）
- 联系电话：公司电话
- 电子邮箱：公司邮箱

**已准备的材料打勾（☑）**：

- [x] 申请人身份证明
- [x] 源代码鉴别材料（前 60 页 + 后 60 页）

## 📎 装订位置

**封面放置位置**：
- 所有材料的最上方
- 申请表之前
- 使用胶装或线装

**装订顺序**（从上到下）：
1. 材料封面
2. 材料目录
3. 软件著作权登记申请表
4. 申请人身份证明
5. 软件说明书
6. 源代码前 60 页
7. 源代码后 60 页

## ✨ 美化建议（可选）

### 添加背景

**方法**：
1. 在 Word 中插入背景图片
2. 选择浅色背景（不影响文字阅读）
3. 设置透明度：80-90%

**推荐背景**：
- 浅色纹理
- 中国风图案
- 青铜器纹样（呼应游戏主题）

### 添加 Logo

**如果有公司/工作室 Logo**：
1. 插入到封面顶部
2. 调整大小：宽度 5-8cm
3. 居中放置

### 添加装饰线

**方法**：
1. 插入 → 形状 → 直线
2. 在标题下方添加装饰线
3. 设置颜色：金色或青铜色
4. 设置粗细：2-3 磅
