## 二、软件功能和技术特点

### 2.1 主要功能

1. **回合制卡牌对战**：实现完整的卡牌对战游戏逻辑，包括回合制流程、法力水晶系统、卡牌打出、随从攻击等核心玩法。

2. **拖拽交互系统**：支持拖拽出牌、拖拽攻击等直观的交互方式，提供流畅的游戏体验。

3. **AI对战系统**：内置智能AI对手，可自动进行出牌、攻击等决策，支持单人游戏模式。

4. **响应式界面**：基于1920×1080设计稿，实现自适应缩放，支持任意分辨率屏幕。

5. **动画系统**：实现攻击动画、伤害数字、过渡动画等视觉效果。

### 2.2 技术特点

1. **前端技术栈**：采用React 18 + TypeScript + Vite + Tailwind CSS现代前端技术栈。

2. **响应式布局**：自研缩放适配系统，基于设计稿尺寸自动计算缩放比例。

3. **状态管理**：采用React Hooks进行细粒度状态管理，优化渲染性能。

4. **动画引擎**：基于requestAnimationFrame实现高性能动画系统。

5. **模块化设计**：组件化开发，代码结构清晰，易于维护和扩展。
