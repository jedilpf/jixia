## 🛠️ 四、截图工具推荐

### 4.1 Windows 自带截图

**快捷键**：`Win+Shift+S`

**模式**：
- 矩形截图（推荐）
- 自由形状截图
- 窗口截图
- 全屏截图

**优点**：
- 系统自带，无需安装
- 快速方便
- 支持延迟截图

---

### 4.2 浏览器开发者工具截图

**Chrome/Edge**：
1. 按 `F12` 打开开发者工具
2. 按 `Ctrl+Shift+P` 打开命令面板
3. 输入"screenshot"
4. 选择：
   - **Capture full size screenshot**（截取整个页面）
   - **Capture screenshot**（截取可视区域）
   - **Capture node screenshot**（截取选定元素）

**优点**：
- 可以截取整个页面（包括滚动区域）
- 高质量 PNG 格式
- 无浏览器边框

---

### 4.3 第三方截图工具

**推荐工具**：

1. **Snipaste**（免费）
   - 官网：https://zh.snipaste.com/
   - 快捷键：F1
   - 功能：截图、贴图、标注

2. **FastStone Capture**（付费）
   - 功能强大
   - 支持滚动截图
   - 自带编辑器

3. **ShareX**（免费开源）
   - 功能最全
   - 支持 GIF 录制
   - 自动上传

## 📐 五、截图技术要求

| **质量** | 高质量（80% 以上） |
| **大小** | 500KB - 2MB |

**必须显示**：
- ✅ 软件名称（谋天下：问道百家）
- ✅ 主要功能界面

## 🎯 十、常见问题

### Q1：截图时浏览器边框去不掉？

**解决方案**：
- 使用浏览器开发者工具截图（Ctrl+Shift+P → Capture full size screenshot）
- 或使用全屏模式（F11）后截图

### Q2：拖拽攻击很难截图？

**解决方案**：
- 使用开发者工具的"Force element state"功能
- 或使用延迟截图（Win+Shift+S → 延迟 3 秒）
- 或请朋友帮忙按鼠标

### Q3：图片太大怎么办？

**解决方案**：
- 使用画图工具调整大小
- 或使用在线压缩工具（tinypng.com）
- 或降低截图分辨率（不低于 1920×1080）

### Q4：截图后插入说明书格式乱了？

**解决方案**：
- 使用 Word 插入图片（更稳定）
- 或调整 Markdown 图片语法
- 或转换为 PDF 后再调整
