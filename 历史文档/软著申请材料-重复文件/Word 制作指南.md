1. **调整字体**：
   - 标题"计算机软件著作权登记申请材料"：黑体，二号，居中
   - "软件名称"等标签：黑体，三号

2. **添加边框**：
   - 插入 → 形状 → 矩形
   - 绘制页面边框
   - 设置线条：黑色，1.5 磅

3. **保存文件**：
   - 文件 → 保存（Ctrl+S）

### 调整格式

Word 打开后，可能需要手动调整格式：

#### 1. 页面设置

- 布局 → 页边距
- 上：2.54cm
- 下：2.54cm
- 左：3.17cm
- 右：3.17cm
- 纸张：A4

#### 2. 字体设置

**标题**：
- 一级标题（如"一、软件概述"）：黑体，三号，加粗
- 二级标题（如"1.1 软件名称"）：黑体，四号，加粗
- 三级标题：宋体，四号，加粗

**正文**：
- 宋体，小四
- 行距：1.5 倍行距

**代码**：
- Consolas 或 Courier New，五号

#### 3. 插入截图

**截图位置**（在 `04-截图/` 文件夹中）：

| 章节 | 截图文件 | 说明 |
|------|----------|------|
| 4.2.4 | 主页大厅界面.png | 主菜单界面 |
| 4.3.5 | 战斗界面.png | 战斗界面 |
| 4.3.5 | （需要手牌特写） | 手牌特写 |
| 4.3.5 | （需要拖拽攻击） | 拖拽攻击 |
| 5.1 | 进入界面.png | 游戏开始 |
| 5.6 | （需要游戏结果） | 游戏结果 |

**插入方法**：
1. 定位到对应章节
2. 插入 → 图片 → 此设备
3. 选择截图文件
4. 调整图片大小（宽度约 15cm）
5. 设置图片格式：居中
6. 添加题注：引用 → 插入题注 → 输入"图 X：说明文字"

#### 4. 添加页眉

1. 插入 → 页眉 → 编辑页眉
2. 输入：`谋天下：问道百家 V1.0`
3. 设置字体：宋体，五号
4. 设置对齐：居中
5. 关闭页眉和页脚

#### 5. 添加页脚页码

1. 插入 → 页码 → 页面底端
2. 选择位置：居中
3. 输入：`第 `
4. 插入 → 页码 → 当前位置
5. 输入：` 页 / 共 `
6. 插入 → 文本 → 快速部件 → 域
7. 选择：NumPages
8. 输入：` 页`
9. 设置字体：宋体，五号
10. 关闭页眉和页脚

#### 6. 保存为 PDF

1. 文件 → 另存为
2. 选择保存位置
3. 文件名：`软件说明书.pdf`
4. 保存类型：**PDF (*.pdf)**
5. 点击"保存"
