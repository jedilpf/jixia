# 软件说明书 PDF 转换指南

**目的**：将 Markdown 格式的说明书转换为 PDF，并添加页眉页脚和截图  
**预计用时**：1 小时

---

## 📄 方法一：使用 Word（推荐）

### 步骤 1：准备 Markdown 文件

**文件位置**：`01-核心提交材料/软件说明书_增强版.md`

### 步骤 2：导入到 Word

**方法 A：直接打开**
1. 打开 Word
2. 文件 → 打开 → 浏览
3. 选择 `软件说明书_增强版.md`
4. Word 会自动转换格式

**方法 B：复制粘贴**
1. 用记事本打开 `软件说明书_增强版.md`
2. 全选（Ctrl+A）
3. 复制（Ctrl+C）
4. 粘贴到 Word

### 步骤 3：调整格式

**页面设置**：
1. 布局 → 页边距
2. 上：2.54cm
3. 下：2.54cm
4. 左：3.17cm
5. 右：3.17cm
6. 纸张：A4

**字体设置**：
- 标题 1：黑体，三号，加粗
- 标题 2：黑体，四号，加粗
- 标题 3：宋体，四号，加粗
- 正文：宋体，小四
- 代码：Consolas，五号

**段落设置**：
- 行距：1.5 倍行距
- 段前：0.5 行
- 段后：0.5 行

### 步骤 4：插入截图

**截图位置**（在说明书中的位置）：

1. **第 4.2.4 节** - 主菜单界面
   - 插入图片：`assets/screenshots/01-main-menu.png`
   - 添加说明：图 1：主菜单界面

2. **第 4.3.5 节** - 战斗界面截图（3 张）
   - 插入图片：`02-battle-view.png`
   - 插入图片：`03-hand-cards.png`
   - 插入图片：`04-drag-attack.png`
   - 添加说明：图 2-4

3. **第 5.1 节** - 游戏开始
   - 插入图片：`05-game-start.png`
   - 添加说明：图 5

4. **第 5.6 节** - 游戏结果
   - 插入图片：`06-game-result.png`
   - 添加说明：图 6

**插入方法**：
1. 定位到对应章节
2. 插入 → 图片 → 此设备
3. 选择截图文件
4. 调整图片大小（宽度约 15cm）
5. 设置图片格式：居中
6. 添加题注：引用 → 插入题注

### 步骤 5：添加页眉

**页眉内容**：`谋天下：问道百家 V1.0`

**添加方法**：
1. 插入 → 页眉 → 编辑页眉
2. 输入：`谋天下：问道百家 V1.0`
3. 设置字体：宋体，五号
4. 设置对齐：居中
5. 勾选"首页不同"（可选）
6. 关闭页眉和页脚

### 步骤 6：添加页脚页码

**页码格式**：`第 X 页 / 共 Y 页`

**添加方法**：
1. 插入 → 页码 → 页面底端
2. 选择位置：居中
3. 输入：`第 `
4. 插入 → 页码 → 当前位置
5. 输入：` 页 / 共 `
6. 插入 → 文本 → 快速部件 → 域
7. 选择：NumPages
8. 输入：` 页`
9. 设置字体：宋体，五号
10. 关闭页眉和页脚

### 步骤 7：另存为 PDF

**保存步骤**：
1. 文件 → 另存为
2. 选择保存位置
3. 文件名：`软件说明书.pdf`
4. 保存类型：PDF（*.pdf）
5. 点击"选项"
6. 勾选"创建后打开文件"
7. 确定 → 保存

## 🌐 方法二：使用在线工具

### 推荐工具

**1. Markdown PDF（网页版）**
- 网址：https://markdown-pdf.com/
- 优点：简单快速
- 缺点：自定义功能少

**使用步骤**：
1. 访问网站
2. 上传 `软件说明书_增强版.md`
3. 配置选项：
   - 纸张大小：A4
   - 页边距：默认
   - 页码：勾选
4. 点击"转换"
5. 下载 PDF

**2. Dillinger（在线 Markdown 编辑器）**
- 网址：https://dillinger.io/
- 优点：实时预览
- 缺点：需要网络

**使用步骤**：
1. 访问网站
2. 导入 Markdown 文件
3. 实时预览效果
4. Export → PDF
5. 下载 PDF

**3. HackMD（协作编辑）**
- 网址：https://hackmd.io/
- 优点：支持协作
- 缺点：需要注册

**使用步骤**：
1. 注册/登录
2. 创建新文档
3. 导入 Markdown 文件
4. 点击"..." → Export → PDF
5. 下载 PDF

## 💻 方法三：使用 VS Code 插件

### 安装插件

**插件名称**：Markdown PDF

**安装步骤**：
1. 打开 VS Code
2. 按 Ctrl+Shift+X 打开扩展
3. 搜索"Markdown PDF"
4. 点击"安装"

### 配置插件

**配置文件位置**：`.vscode/settings.json`

**配置内容**：
```json
{
  "markdown-pdf.format": {
    "format": "A4",
    "margin": {
      "top": "2.54cm",
      "bottom": "2.54cm",
      "left": "3.17cm",
      "right": "3.17cm"
    },
    "header": {
      "height": "1cm",
      "contents": [
        {
          "type": "text",
          "text": "谋天下：问道百家 V1.0",
          "style": {
            "font-size": "10pt",
            "text-align": "center"
          }
        }
      ]
    },
    "footer": {
      "height": "1cm",
      "contents": [
        {
          "type": "text",
          "text": "第 {{page}} 页 / 共 {{pages}} 页",
          "style": {
            "font-size": "10pt",
            "text-align": "center"
          }
        }
      ]
    }
  }
}
```

### 转换 PDF

**步骤**：
1. 在 VS Code 中打开 `软件说明书_增强版.md`
2. 按 Ctrl+Shift+P
3. 输入"Markdown PDF: Export (pdf)"
4. 回车
5. 等待转换完成
6. PDF 文件会生成在同目录

## 🖨️ 方法四：使用 Pandoc（专业工具）

### 安装 Pandoc

**下载地址**：https://pandoc.org/installing.html

**安装步骤**：
1. 下载安装包
2. 运行安装程序
3. 按提示安装
4. 重启终端

### 转换命令

**基本命令**：
```bash
pandoc 软件说明书_增强版.md -o 软件说明书.pdf
```

**带页眉页脚的命令**：
```bash
pandoc 软件说明书_增强版.md \
  -o 软件说明书.pdf \
  --pdf-engine=xelatex \
  -V header-includes="\usepackage{fancyhdr}" \
  -V header-includes="\pagestyle{fancy}" \
  -V header-includes="\fancyhead[C]{谋天下：问道百家 V1.0}" \
  -V header-includes="\fancyfoot[C]{第 \thepage 页 / 共 \pageref{LastPage} 页}" \
  -V colorlinks=true
```

## 🔧 常见问题

### Q1：转换后格式乱了？

**解决方案**：
- 使用 Word 手动调整
- 或在 Markdown 中使用更简单的语法
- 避免复杂的表格和公式

### Q2：图片无法显示？

**解决方案**：
- 检查图片路径是否正确
- 使用绝对路径
- 或将图片嵌入 Markdown

### Q3：页眉页脚不显示？

**解决方案**：
- 检查 PDF 转换工具设置
- 确保勾选了页眉页脚选项
- 或使用 Word 手动添加

### Q4：PDF 文件太大？

**解决方案**：
- 压缩图片（tinypng.com）
- 降低图片分辨率
- 或使用 PDF 压缩工具
