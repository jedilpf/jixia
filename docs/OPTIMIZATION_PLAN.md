# 性能与稳定性优化计划（v2）

## 1. 目标

- 对局链路可复现、可回放、可审计
- 明辩/暗谋并发输入稳定
- 五层结算耗时可控
- 客户端帧率与网络抖动下仍可用

## 2. 技术优先级

### P0

- 阶段状态机幂等化
- 操作锁定协议与超时兜底
- 结算日志结构化（可回放）
- 异常对局自动上报（阶段/事件/版本）

### P1

- 三席结算批处理优化
- 动画与结算线程解耦
- 论场效果缓存与按需计算
- 首屏资源拆包与懒加载

### P2

- 回放增量存储
- 大盘指标聚合任务优化
- 观战模式性能预研

## 3. 验收指标

- 结算阶段 P95 耗时
- 阶段切换失败率
- 操作锁定冲突率
- 回放一致性通过率
- 客户端崩溃率

## 4. 回归场景

- 双方最后 1 秒同时提交
- 暗策揭示与断线重连
- 连续触发论场技边界条件
- 三席同时击破与跨层传递
