# 知识库-01-项目总览

**文档版本**: 2.1.0  
**更新日期**: 2026-03-09  
**基线**: 今日确认的 160 张设定集（含战斗设定）

---

## 1. 项目定位

《百家争鸣》是一款同步博弈卡牌对战游戏，核心在于：

- 明辩/暗谋双阶段信息对抗
- 三席双层站位经营
- 着书带来的中后期节奏兑现
- 论场分流带来的构筑生态差异

---

## 2. 统一规则摘要

- 匹配前：选卡组 + 选论场
- 回合：明辩 10 秒 -> 暗谋 10 秒 -> 揭示 -> 五层结算
- 战场：先声席 / 主辩席 / 余论席；每席前席/后席
- 传递：前席 -> 后席 -> 主辩者
- 资源：心证、灵势、护印、证立、失序、文脉
- 牌类：立论、策术、反诘、门客、玄章

---

## 3. 首发论场

- 稷下学宫：标准环境
- 火德论坛：前线压制环境
- 藏书秘阁：着书成长环境
- 玄机观星台：暗策信息差环境

---

## 4. 当前研发状态

- 新战斗规则已确定并文档化
- 代码侧仍有历史原型待迁移
- 后续开发以知识库和 v1 战斗文档为唯一准入

---

## 5. 关键目标

1. 做实同步回合体验  
2. 做实三席与着书价值  
3. 做实四论场分流生态  
4. 把 160 张设定集稳定落地为可持续卡池
