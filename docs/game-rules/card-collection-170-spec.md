# 稷下学宫卡牌大全（170张）

## 一、概述

本文档定义了稷下学宫卡牌游戏的完整卡牌体系，共 **170张卡牌**，分布在 **16个特色门派** 和 **通用** 类别中。

### 1.1 核心机制

| 概念 | 含义 | 说明 |
|------|------|------|
| **立论** | 人（角色/单位） | 有辩才(attack)和学识(hp)，持续存在于战场 |
| **策术** | 法术牌 | 即时生效的一次性效果，无持续属性 |
| **底蕴** | 费用 | 使用卡牌消耗的资源 |
| **辩才** | 攻击力 | 立论对其他卡牌造成伤害的能力 |
| **学识** | 血量 | 立论的生命值，归零则退场 |

### 1.2 等级系统（杠数显示）

| 稀有度 | 等级 | 杠数显示 | 颜色 |
|--------|------|----------|------|
| 常见 | 一等 | \| | 灰色 #9ca3af |
| 稀有 | 二等 | \|\| | 蓝色 #60a5fa |
| 史诗 | 三等 | \|\|\| | 紫色 #a855f7 |
| 传说 | 三等 | \|\|\|\| | 金色 #f59e0b |

### 1.3 数值设计公式

**立论卡牌**：
- 常见：辩才 = 费用，学识 = 费用 + 1
- 稀有：辩才 = 费用，学识 = 费用 + 2
- 史诗：辩才 = 费用 + 1，学识 = 费用 + 2
- 传说：辩才 = 费用 + 2，学识 = 费用 + 2

**策术卡牌**：
- 无辩才和学识属性
- 根据效果强度调整费用

## 五、设计原则

### 5.1 平衡性原则

1. **费用曲线**：低费卡牌占比60%，确保游戏前期有牌可出
2. **类型平衡**：立论和策术各占50%，兼顾场面和节奏
3. **稀有度分布**：常见60%、稀有27%、史诗10%、传说3%，保证收集梯度

### 5.2 门派特色

每个门派有明确的主题定位：
- **攻击型**：九阵堂、天工坊
- **防御型**：礼心殿、玄匠盟
- **控制型**：衡戒廷、名相府、筹天阁
- **回复型**：归真观、万农坊、杏林馆
- **干扰型**：稗言社
- **资源型**：游策阁、兼采楼
- **特殊型**：司天台（判定）、两仪署（平衡）、养真院（成长）

### 5.3 卡牌命名规范

1. **立论**：多用人物、职位、状态（如"司史执笔"、"锋将·破军"）
2. **策术**：多用典籍、器物、行为（如"铁券禁令"、"云梯凌虚"）
3. **背景描述**：4-6字概括，体现门派特色
4. **技能描述**：简洁明了，避免歧义
