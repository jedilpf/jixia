# 卡牌术语规范文档

## 一、核心术语（必须使用）

### 1.1 卡牌类型
| 术语 | 含义 | 使用场景 |
|------|------|----------|
| **立论** | 人（角色/单位） | 有辩锋和学识，持续存在于战场 |
| **策术** | 法术牌（一次性效果） | 即时生效，无持续属性 |

### 1.2 资源与属性
| 术语 | 含义 | 使用场景 |
|------|------|----------|
| **势** | 费用/资源 | 使用卡牌消耗的资源 |
| **辩锋** | 攻击力 | 立论对其他卡牌造成伤害的能力 |
| **学识** | 血量/生命值 | 立论的生命值，归零则退场 |
| **护体** | 护盾 | 临时生命值，可抵挡伤害 |

### 1.3 状态效果
| 术语 | 含义 | 使用场景 |
|------|------|----------|
| **惑** | 负面状态 | 失去准确目标，攻击可能打错 |
| **禁言** | 负面状态 | 不能使用策术 |
| **正义** | 正面状态 | 下回合伤害增加一点 |
| **清静** | 正面状态 | 受到伤害减少 |
| **轻盈** | 正面状态 | 闪避伤害 |
| **失序** | 负面状态 | 不能使用立论 |

## 二、通用动作术语

### 2.1 基础动作
- **抽牌** / **抽一张牌** / **抽两张牌** / **抽三张牌**
- **弃牌** / **弃置一张牌**
- **回复**（势）
- **造成伤害**
- **获得**（状态/护体）
- **净化**（负面状态）
- **反制**（打断敌方策术）
- **闪避**（躲避攻击）
- **登场**（立论进场时触发）
- **被动**（持续生效的效果）

### 2.2 位置相关
- **主议** / **旁议**（战场位置）
- **牌库** / **牌库顶**
- **手牌**
- **弃牌堆**

| 【场地】 | 策术（场地） |

## 四、描述规范

### 4.1 背景描述
- 4-8字为宜
- 体现门派特色
- 避免生僻字

**示例**：
- 温言缓语，以礼压躁。（礼心殿）
- 闭目观云，心静尘退。（归真观）
- 鹤行无声，步落虚空。（归真观）

### 4.2 技能描述
- 使用标准数字（一、二、三、四、五、六、七、八、九、十）
- 避免【】符号
- 主语明确（你、敌方、友方）
- 条件清晰（若...则...）

**示例**：
- 抽一张牌；若本回合已使用过礼心殿牌，改为抽两张牌。
- 回复三点势；若本回合未受伤，额外抽一张牌。
- 敌方使用立论时受一点伤害；本回合已受伤则额外加一点。

## 五、新增术语申请流程

如需新增术语，需满足：
1. 有明确的 gameplay 意义
2. 不与现有术语冲突
3. 在文档中定义清楚
4. 经确认后加入本规范
