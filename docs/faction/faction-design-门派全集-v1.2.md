## 二、各门派详细设定

### 2.1 礼心殿（儒家）

**学派**：儒家
**定位**：辅助/成长
**核心机制**：立论、证立、着书

**设计理念**：
以"仁"为核心，强调立言有据、通过论证积累优势。适合喜欢经营感的玩家。

**代表卡牌**：

| 牌名 | 类型 | 用度 | 辩锋 | 根基 | 效果 |
|------|------|------|------|------|------|
| 温言立论 | 立论 | 1 | 1 | 1 | 若置于主议，本回合 +1 辩锋。 |
| 竹牯抄录 | 施策 | 2 | - | - | 抽2张牌，然后弃1张牌；若你在主议有立论，该立论恢复1根基。 |
| 讲席清规 | 施策 | 2 | - | - | 选择一张己方立论：本回合 +1 辩锋。 |
| 司史执笔 | 施策 | 3 | - | - | 选择一张己方立论：其本回合第一次受到敌方施策指定时，该施策对其无效。 |
| 礼辩同归 | 立论 | 5 | 3 | 3 | 回合结束时，若你赢得主议，选择一张己方立论恢复1根基。 |
| 执礼门生 | 立论 | 2 | 2 | 2 | 若你本回合打出过至少1张施策，本牌本回合 +1 辩锋。 |
| 正席问礼 | 施策 | 2 | - | - | 选择一张己方立论：本回合其辩锋 +1。 |
| 大成议官 | 立论 | 5 | 2 | 2 | 进场时，若置于主议，对位敌方立论本回合 -1 辩锋。 |

---

### 2.2 名相府（法家）

**学派**：法家
**定位**：控制/诡辩
**核心机制**：反诘、文本失效、削弱

**设计理念**：
以"法"为核心，强调规则利用和对手削弱。适合喜欢干扰对手的玩家。

**代表卡牌**：

| 牌名 | 类型 | 用度 | 辩锋 | 根基 | 效果 |
|------|------|------|------|------|------|
| 白马非马 | 施策 | 1 | - | - | 选择一张敌方立论：其文本失效直到回合结束；若其在主议，再 -1 辩锋。 |
| 名实互诘 | 施策 | 2 | - | - | 选择一张敌方立论；若其本回合获得过辩锋提升，则该立论本回合辩锋改为1；否则其本回合 -1 辩锋。 |
| 同异之辩 | 施策 | 2 | - | - | 二选一：1）一张己方立论本回合 +1 辩锋；2）一张敌方立论本回合 -1 辩锋。 |
| 辞锋术士 | 立论 | 2 | 2 | 3 | 每当你打出施策，本牌本回合 +1 辩锋。 |
| 诡辩连环 | 施策 | 4 | - | - | 选择至多两张不同立论；它们本回合各 -1 辩锋；若其中有立论本回合已文本失效，再抽1张牌。 |
| 名辩成势 | 立论 | 3 | 3 | 3 | 进场时，选择一张敌方立论；本回合它不能获得辩锋提升。若其在主议，本牌本回合 +1 辩锋。 |
| 归谬设问 | 施策 | 2 | - | - | 选择一张敌方立论；本回合 -2 辩锋；若其本回合已文本失效，则改为 -3 辩锋。 |
| 名实归真 | 立论 | 4 | 2 | 3 | 若你本回合使敌方立论文本失效，则你赢得主议时，对位敌方立论额外失去1根基。 |

### 2.3 游策阁（纵横家）

**学派**：纵横家
**定位**：机动/易席
**核心机制**：易席、位移、灵活切换

**设计理念**：
以"势"为核心，强调战场机动性和策略灵活性。适合喜欢多变的玩家。

**代表卡牌**：

| 牌名 | 类型 | 用度 | 辩锋 | 根基 | 效果 |
|------|------|------|------|------|------|
| 合纵之策 | 施策 | 1 | - | - | 将一张己方立论移到另一个议位；若它移入旁议，抽1张牌，然后弃1张牌。 |
| 连横之契 | 施策 | 2 | - | - | 交换你主议与旁议上的立论位置。 |
| 游说之辞 | 施策 | 2 | - | - | 选择一张敌方立论：本回合 -2 辩锋；若你的大势落后，再抽1张牌，然后弃1张牌。 |
| 盟书 | 立论 | 2 | 2 | 4 | 若你赢得旁议，另一张己方立论本回合 +1 辩锋；若该立论在主议，再抽1张牌，然后弃1张牌。 |
| 游士借席 | 立论 | 1 | 2 | 2 | 每当本牌发生易席，本回合 +1 辩锋。 |
| 八面逢源 | 立论 | 4 | 3 | 3 | 进场时，移动任意一张立论；若因此改变某个议位的胜负局面，本牌获得 +1 根基。 |
| 说客回函 | 施策 | 3 | - | - | 从弃牌堆拿回1张施策到手牌；若你本回合发生过易席，改为拿回2张，再弃1张。 |
| 纵横门生 | 立论 | 2 | 2 | 3 | 你每回合第一次施策 -1 用度；若本牌在旁议，每回合第一次有己方立论发生易席时，本牌本回合 +1 辩锋。 |

## 三、门派设计原则

### 3.1 核心原则

1. **门派特色鲜明**：每个门派应有独特的机制和玩法
2. **门派平衡**：各门派胜率应接近50%
3. **配合多样**：门派之间应有多种配合可能
4. **易于上手**：新手能快速理解门派特色

### 3.2 门派定位分布

| 定位类型 | 特点 | 代表门派 |
|----------|------|----------|
| 攻击型 | 高辩锋、快节奏 | 火德论坛适配 |
| 防御型 | 高根基、持久战 | 万农坊 |
| 控制型 | 削弱对手、干扰节奏 | 名相府 |
| 机动型 | 灵活变换、适应性强 | 游策阁 |
| 辅助型 | 增益友方、配合为主 | 礼心殿 |
