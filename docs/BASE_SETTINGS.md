## 一、卡牌类型

| 类型 | 英文ID | 说明 |
|------|--------|------|
| **立论** | argument | 核心论点牌，直接对决 |
| **策术** | scheme | 策略事件牌，触发效果 |
| **玄章** | support | 场地/装备牌，持续影响 |
| **门客** | character | 角色牌，有攻击和底蕴 |
| **反诘** | counter | 反制牌，打断对手 |

### 玄章子类型
- **装备**：给门客装备，提供增益
- **场地**：全场持续效果

---

## 二、核心效果

| 效果 | 格式 | 说明 |
|------|------|------|
| **护持** | 【护持N】 | 防御护盾，吸收N点伤害 |
| **学识** | 【学识+N】 | 积累型增益，达到阈值触发效果 |
| **失序** | 【失序+N】 | 负面状态，行动受限 |
| **沉默** | 【沉默（N回合）】 | 无法使用技能 |
| **怀疑** | 【怀疑+N（N回合）】 | 负面标记，影响判定 |

## 五、游戏数值

### 核心参数
```typescript
{
  victoryMomentum: 8,      // 胜利所需大势
  maxRounds: 20,           // 最大回合数
  initialHandSize: 5,      // 初始手牌数
  actionLimitPerTurn: 3,   // 每回合最多出牌数
  maxCardsPerZone: 3,      // 每个议区最多卡牌数
  deckSize: 24,            // 牌组大小
  manaByRound: [2,3,4,5],  // 各回合势上限
}
```

### 势曲线
| 回合 | 1 | 2 | 3 | 4+ |
|------|---|---|---|-----|
| 势上限 | 2 | 3 | 4 | 5 |

## 六、术语对照表

### 新旧术语对照
| 旧术语 | 新术语 | 说明 |
|--------|--------|------|
| 技能 | 立论 | 核心论点牌 |
| 事件 | 策术 | 策略事件牌 |
| 场地 | 玄章（场地型） | 持续效果牌 |
| 装备 | 玄章（装备型） | 可装备牌 |
| 角色 | 门客 | 有攻/血的角色牌 |
| 反制 | 反诘 | 打断牌 |
| 清明/清静 | 学识 | 积累型增益 |

### 保留术语
- **装备：** 玄章牌的装备效果标签
- **场地：** 玄章牌的场地效果标签
- **装备者** 被装备的目标门客
- **技能** 门客的能力（非卡牌类型）

## 七、数据结构

### CardData 接口
```typescript
interface CardData {
  id: string;           // 唯一标识
  name: string;         // 卡牌名称
  faction: string;      // 所属门派
  type: string;         // 立论/策术/玄章/门客/反诘
  rarity: string;       // 常见/稀有/史诗/传说
  cost: number;         // 费用
  attack?: number;      // 攻击（门客牌）
  hp?: number;          // 底蕴（门客牌）
  shield?: number;      // 护盾（玄章牌）
  skill: string;        // 技能描述
  background: string;   // 背景故事
  unlockTier?: number;  // 解锁等级 1/2/3
}
```
