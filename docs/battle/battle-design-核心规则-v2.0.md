# 《思筹之录》核心战斗规则 v2.0

## 文档说明

本文档为 v2.0 版本，包含以下核心变更：
- **卡牌数量**：160 张 → 170 张
- **结算机制**：五层结算 → 两层结算
- **出牌流程**：明辩/暗谋 → 出牌→结算循环

## 二、游戏流程（v2.0 简化版）

### 2.1 两层结算机制

```
┌─────────────────────────────────────────────────────────────┐
│                        回合开始                              │
├─────────────────────────────────────────────────────────────┤
│  1. 回合开始阶段                                             │
│     - 双方获得用度                                          │
│     - 刷新用度                                              │
├─────────────────────────────────────────────────────────────┤
│  2. 第一层：出牌 → 结算                                      │
│     - 玩家出牌                                              │
│     - 立即结算该牌效果                                      │
│     - 即时反馈，无等待                                      │
├─────────────────────────────────────────────────────────────┤
│  3. 第二层：出牌 → 结算                                      │
│     - 玩家再次出牌                                          │
│     - 再次结算该牌效果                                      │
│     - 在范围内循环                                          │
├─────────────────────────────────────────────────────────────┤
│  4. 回合结束                                                 │
│     - 弃牌（超过手牌上限）                                  │
│     - 抽牌至手牌上限                                        │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 简化流程特点

**与传统五层结算的区别：**

| 对比项 | 五层结算（旧） | 两层结算（新） |
|--------|---------------|---------------|
| 等待时间 | 明辩10秒+暗谋10秒 | 即时结算 |
| 反馈速度 | 延迟揭示 | 立刻看到结果 |
| 操作复杂度 | 明牌/暗牌分开选 | 出牌即结算 |
| 游戏节奏 | 较慢 | 更流畅 |

### 2.3 议区结构

每位玩家有 **两个议区**：

| 议区 | 说明 | 特点 |
|------|------|------|
| 主议 | 中央主要议区 | 决定大势走向 |
| 旁议 | 侧翼辅助议区 | 获得筹等资源 |

**规则**：
- 每个议区最多放置 3 张牌
- 主议胜获得 +1 大势
- 旁议胜获得 1 个筹（可用来降低出牌费用）

## 三、卡牌类型说明

### 3.1 立论

立论是放置在场上的单位牌，类似于传统卡牌游戏中的"生物"。

**特点**：
- 有辩锋（攻击力）
- 有根基（生命值）
- 可以被施策指定
- 可以发生易席

### 3.2 施策

施策是即时生效的咒语牌，类似于传统卡牌游戏中的"法术"。

**特点**：
- 立即生效
- 执行后进入弃牌堆
- 可以指定立论为目标
- 部分有持续效果

**只有这两种卡牌类型。不要添加反诘、门客、玄章等额外类型。**

---

## 四、重要概念解释

### 4.1 辩锋

辩锋相当于卡牌的"攻击力"，决定立论在争鸣中的胜负。

### 4.2 根基

根基相当于卡牌的"生命值"，归零时立论离场。

### 4.3 用度

用度相当于"法力费用"，用于支付打出卡牌的代价。

### 4.4 大势

大势是游戏的核心资源，率先达到8点即获胜。

### 4.5 筹

旁议胜可获得1个筹，最多持有1个。消耗筹可令本回合出牌-1用度。

### 4.6 护印

护印可以吸收伤害，保护立论或主辩者。
