# 首发战斗界面框架 v1

**日期**: 2026-03-09  
**实现状态**: 已接入前端骨架页（`src/components/BattleFrameV2.tsx`）

---

## 1. 设计目标

基于当前战斗机制，把界面拆成可独立迭代的五层：

1. 顶部战局层（回合、阶段、倒计时、议题、论场）
2. 中部战场层（三席双层核心）
3. 左侧资源层（双方核心资源）
4. 右侧信息层（论场效果、短日志）
5. 底部操作层（主论/应对/暗策/着书 + 手牌）

---

## 2. 区域与机制映射

### 2.1 顶部战局层

- 显示阶段流：明辩 -> 暗谋 -> 揭示 -> 结算
- 显示当前阶段剩余秒数
- 显示当前议题和回合数
- 显示当前论场与核心效果摘要

### 2.2 中部战场层

- 三列固定：先声席 / 主辩席 / 余论席
- 每列四格展示：敌后、敌前、我前、我后
- 顶部标注伤害传递顺序：前席 -> 后席 -> 主辩者

### 2.3 底部操作层

- 主论位（公开）
- 应对位（公开）
- 暗策位（隐藏）
- 着书位（明辩阶段）
- 手牌行（类型 + 费用 + 名称）
