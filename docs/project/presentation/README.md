# 《思筹之录》PPT 文档入口（主线）

> 目标：把 PPT 相关文档集中在一个固定入口，方便项目负责人与 AI 快速定位。

## 唯一主入口

- 主入口目录：`docs/project/presentation/`
- 主线构思稿：`docs/project/presentation/ppt-mainline-outline.md`

## 当前结构

- `README.md`：本入口说明（先看这里）
- `ppt-mainline-outline.md`：11 页主线介绍稿构思（可直接用于成稿）
- `PORTFOLIO.md`：**KMD 评审入口文档（价值优先）**（清楚定义、审美立场、研究问题、原型证据、未来路径）

技术细节（架构、规则实现、模块级说明）建议放在专门技术文档中，不堆叠在评审入口页。

## 使用规则

1. 与《思筹之录》对外介绍相关的 PPT 文档，优先放在本目录。
2. 新增文档时保持“用途清晰 + 文件名可读”，避免同主题散落到多个目录。
3. 若必须调整路径，先在旧路径保留“迁移指引”，再更新索引入口。

## 维护说明

- 当前维护口径：主线可运行 + 持续开发
- 未确认数据统一写 `UNKNOWN`，避免在演示材料中硬填数值
