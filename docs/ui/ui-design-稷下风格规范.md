# 视觉规范：青铜机关风格（v2）

**目标**: 服务“明辩/暗谋 + 三席布阵 + 论场差异”的信息表达。

---

## 1. 视觉方向

- 主材质：青铜、竹简、墨迹、铜环仪
- 情绪：庄重、智性、博弈感，不做轻浮卡通感
- 信息优先级：阶段信息 > 席位信息 > 数值信息 > 装饰信息

---

## 2. 色彩 Token（建议）

- `--bg-deep`: `#11161D`
- `--bronze`: `#8B7355`
- `--bronze-light`: `#B39A79`
- `--jade`: `#3D6E63`
- `--ember`: `#D56A1A`
- `--paper`: `#D8CCB1`
- `--ink`: `#1A1A1A`

---

## 3. 核心界面区块

- 顶部：回合号、阶段计时、议题与论场状态
- 中部：三席双层战场（必须清晰区分前席与后席）
- 底部：手牌区与明辩/暗谋操作区
- 右侧：本局日志与结算提示（短提示，不弹长窗）

---

## 4. 动效规则

- 明辩结束 -> 暗谋开始：阶段条颜色切换 + 轻量音效
- 暗谋揭示：卡牌翻开动画 200~300ms
- 五层结算：分层闪示，不并发堆叠
- 穿席伤害：沿“前席 -> 后席 -> 主辩者”做单向轨迹

---

## 5. 文本排版

- 标题：思源宋体或同类衬线中文字体
- 数值：等宽字体以减少跳动
- 卡牌描述：限制行高和字重，优先可读性
- 禁用多余修饰词，避免术语噪音

---

## 6. 易用性要求

- 10 秒阶段倒计时必须一眼可见
- 明辩/暗谋可操作区域必须视觉分区明确
- 暗策“已放置/未放置”状态必须显著
- 三席是否被击破必须有明确状态标识
