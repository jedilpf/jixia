## 5. 使用规则（避免再次文档失控）

1. 新文档必须先判断是否属于“主线必读 / 专项 / 归档”。
2. 同一主题优先更新现有文档，不重复开新文件。
3. 任何包含桌面路径、跨仓扫描、破坏性清理命令的文档，默认标注为“需校正后再执行”。
4. 每周至少一次更新 [本周文档收口清单](weekly-docs-curation-20260326.md)。
