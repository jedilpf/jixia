## 3. 技术约束 (Technical Constraints)

1. **禁止硬编码**：所有 UI 字符串应优先从 `V9_CONSTANTS` 或特定国际化/配置项中读取。
2. **图标对齐**：若按钮使用了 `IconBambooSlips`，则内部标题必须配以同样的装饰意象。
3. **面包屑同步**：返回按钮的文本（Back）应统一为“返回 [上一级模块名]”，例如“返回百家文库”。
