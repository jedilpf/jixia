# 《思筹之录》游戏设计规格书

**状态**：v1.0（待评审）
**最后更新**：2026-05-03

---

## 一、游戏目标

**胜利条件**：率先积累 **8 大势** 者获胜

游戏是玩家 vs AI 的 1v1 对战卡牌游戏。每回合通过出牌争夺议区，赢取大势。

---

## 二、资源（6种，不可增减）

| 资源 | 用途 | 界面显示 |
|------|------|---------|
| **势** | 出牌费用 | TopStatusBar 蓝/绿数字 |
| **大势** | 胜利积分，先到8赢 | TopStatusBar 大势 |
| **筹** | 出牌时自动-1费用（最多1个） | TopStatusBar 筹 |
| **辩锋** | 攻击力（立论牌的面板数值） | CardSlot 内显示 |
| **学识** | 生命值（立论牌的面板数值） | CardSlot 内显示 |
| **护体** | 护盾，抵消伤害 | TopStatusBar 护印 |

---

## 三、卡牌（仅2种）

| 类型 | 行为 | 场上存在 |
|------|------|---------|
| **立论** | 放置到议区，持续在场 | ✅ 留在议区 |
| **策术** | 放置时**立即触发**效果 | ❌ 不留场 |

**测试卡池**：6张立论 + 6张策术，共12张

---

## 四、完整游戏循环（每回合）

```
【第N回合】
  1. play_1 阶段（40秒）
      - 玩家选1张牌出到 主议 或 旁议
      - AI自动出1张牌
      - 双方都锁定 → 进入 resolve_1

  2. resolve_1 阶段（立即）
      - 策术立即触发效果
      - 立论进入议区

  3. play_2 阶段（40秒）
      - 同 play_1，可出第2张

  4. resolve_2 阶段（立即）
      - 同 resolve_1

  5. 议区结算
      - 主议：辩锋高者 +1 大势
      - 旁议：辩锋高者 +1 筹（上限1）
      - 平局：各 -1 学识

  6. 抽牌阶段
      - 每方补手牌到 5 张

  7. 进入第 N+1 回合
```

## 五、出牌操作（当前已实现，点击模式）

### 步骤1：选牌
- 点击手牌区的一张卡 → 卡牌高亮选中

### 步骤2：选座位
- 点击"主议"或"旁议" → 座位高亮

### 步骤3：确认出牌
- 点击"确认出牌"按钮 → 牌出到议区

### 步骤4：锁定回合
- 点击"结束回合" → 锁定当前层，进入下一阶段

**策术**：出牌时**立即触发**，动画显示效果

## 六、议区与容量规则

- **主议（zhu_yi）**：每次胜利 +1 大势
- **旁议（pang_yi）**：每次胜利 +1 筹
- 每个议区最多 **3 张卡**
- 立论在场上直到被消灭
- 策术触发后消失

## 八、AI 对手行为

AI 在 `play_1` 和 `play_2` 阶段自动出牌：
- 优先选高辩锋/高学识的立论
- 优先选高伤害/高抽牌的策术
- 出牌后**立即锁定**（玩家需要自己锁定）
- AI 行为静默，不显示动画（后续改进可加）

## 九、当前 UI 问题与改进方案

### 🔴 P0 - 阻塞体验（必须修复）

#### 问题1：手牌区没有标题
**现象**：用户不知道底部是手牌
**修复**：在手牌区左上角加"手牌"标签

#### 问题2：议区没有操作引导
**现象**：用户不知道点击座位可以出牌
**修复**：在座位枢纽（"议"/"旁"图标）旁边加文字提示

#### 问题3：操作流程不清晰
**现象**：用户不知道"选牌→选座位→确认"三步
**修复**：BottomControls 中间按钮区显示当前步骤提示

### 🟡 P1 - 重要体验（应该修复）

#### 问题4：出牌后没有反馈
**现象**：点击确认后，卡牌消失，没有明显动画
**修复**：加简单的卡牌移出动画 + 费用飘字

#### 问题5：策术触发没有明显提示
**现象**：策术立即触发，但没有视觉强调
**修复**：策术触发时加闪光/飘字效果

#### 问题6：阶段切换不清晰
**现象**：resolve_1 到 play_2 切换时，用户不知道发生了什么
**修复**：加 PhaseTransition 遮罩动画（已有组件，确认启用）

### 🟢 P2 - 优化体验（可以后续做）

#### 问题7：拖拽出牌体验
- 当前是点击模式，拖拽是更直觉的交互
- **不紧急**：点击模式完全可玩

#### 问题8：AI 出牌不透明
- AI 静默自动出牌
- **后续改进**：AI 出牌时在对方 CardSlot 短暂显示卡牌背面动画
