**先到 8 大势者获胜**

### 1.3 用度曲线

| 回合 | 最大用度 |
|------|----------|
| 第1轮 | 2 |
| 第2轮 | 3 |
| 第3轮 | 4 |

## 二、游戏流程

### 2.1 回合结构

```
┌─────────────────────────────────────────────────────────────┐
│                        回合开始                              │
├─────────────────────────────────────────────────────────────┤
│  1. 回合开始阶段                                             │
│     - 双方获得用度                                          │
│     - 刷新用度                                              │
├─────────────────────────────────────────────────────────────┤
│  2. 明辩阶段                                                │
│     - 双方同步选择明牌                                      │
│     - 选择目标议区                                          │
│     - 等待确认/倒计时结束                                   │
├─────────────────────────────────────────────────────────────┤
│  3. 暗谋阶段                                                │
│     - 双方同步选择暗牌（可选）                              │
│     - 暗牌不显示给对手                                      │
├─────────────────────────────────────────────────────────────┤
│  4. 揭示阶段                                                │
│     - 同时翻开明牌和暗牌                                    │
│     - 计算辩锋                                               │
├─────────────────────────────────────────────────────────────┤
│  5. 结算阶段                                                │
│     - 判定胜负                                               │
│     - 执行效果                                               │
│     - 分配大势/筹                                           │
├─────────────────────────────────────────────────────────────┤
│  6. 回合结束                                                │
│     - 弃牌（超过手牌上限）                                  │
│     - 抽牌至手牌上限                                        │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 议区结构

每位玩家有 **两个议区**：

| 议区 | 说明 | 特点 |
|------|------|------|
| 主议 | 中央主要议区 | 决定大势走向 |
| 旁议 | 侧翼辅助议区 | 获得筹等资源 |

**规则**：
- 每个议区最多放置 3 张牌
- 主议胜获得 +1 大势
- 旁议胜获得 1 个筹（可用来降低出牌费用）

## 三、三层战斗结构

### 3.1 第一层：表层战斗

- 出牌
- 比拼主议 / 旁议
- 获得大势
- 场上单位离场或存活

### 3.2 第二层：明辩

- 双方同步翻开本轮打出的牌
- 玩家能够看见对方本轮具体打了什么牌
- 这些牌代表对当前议题的公开立场和观点

### 3.3 第三层：暗辩

- 在揭示前，玩家只能看到对方"出了没有、出了几张、放在哪个议区"
- 看不到对方具体牌名
- 暗辩的本质不是隐藏数值，而是**隐藏本轮真实选择与组合意图**

## 四、议题系统

### 4.1 议题概述

议题不是开局固定 Buff，也不是局前配置项。  
议题是战斗中间的 **公共争夺核心**。

一局战斗只有 **1 个中央议题**。

### 4.2 议题阶段

1. **议题种子**：刚开始，只是一个模糊议题
2. **议题争夺**：双方出牌不断推动议题方向
3. **议题引爆**：每隔固定轮次判定一次，若条件达成则触发高光演出和强战斗效果

## 五、论场系统

### 5.1 论场定义

论场是玩家在匹配前主动选择的公共规则环境。  
双方选择相同论场后，才会进入同一匹配池。

### 5.2 论场结构

每个论场包含：
- 1 个基础被动
- 1 个偏向效果
- 1 个论场技

### 5.3 首发四大论场

#### 论场一：稷下学宫（标准论场）

| 类型 | 名称 | 效果 |
|------|------|------|
| 基础被动 | 公议初启 | 双方开局各获得 1 护印 |
| 偏向效果 | 立言有据 | 每回合第一次打出立论时，若本回合未着书，则获得 1 证立 |
| 论场技 | 公议 | 本局一次。令你本回合第一张反诘牌费用 -1 |

**适配门派**：礼心殿、名相府、游策阁、通用构筑

#### 论场二：火德论坛（快攻论场）

| 类型 | 名称 | 效果 |
|------|------|------|
| 基础被动 | 烈辩先声 | 前席门客争鸣时辩锋 +1 |
| 偏向效果 | 穿席余烬 | 一席前席被击破后，若该路仍有剩余伤害，则对主辩者额外造成 1 心证伤害 |
| 论场技 | 焚势 | 本局一次。选择一席，本回合该席伤害不能被护印完全吸收 |

**适配门派**：九阵堂、衡戒廷、游策阁进攻构筑

#### 论场三：藏书秘阁（控制论场）

| 类型 | 名称 | 效果 |
|------|------|------|
| 基础被动 | 秘阁藏卷 | 本局第一次着书后，额外获得 1 护印 |
| 偏向效果 | 文脉深藏 | 你的文脉上限 +1（或第2次着书开始，每次着书额外回复1心证） |
| 论场技 | 校书 | 本局一次。立即执行一次着书，然后抽1，弃1 |

**适配门派**：礼心殿、万农坊、归藏台

#### 论场四：玄机观星台（信息论场）

| 类型 | 名称 | 效果 |
|------|------|------|
| 基础被动 | 星轨照影 | 双方暗策锁定后，可看到对方暗策的牌类 |
| 偏向效果 | 天机偏衡 | 每回合第一次改写议题时，抽1 |
| 论场技 | 窥衡 | 本局一次。查看对方本回合暗策的具体内容，且你本回合第一张反诘效果 +1 |

**适配门派**：司天台、名相府、游策阁

## 七、卡牌类型说明

### 7.1 立论

立论是放置在场上的单位牌，类似于传统卡牌游戏中的"生物"。

**特点**：
- 有辩锋（攻击力）
- 有根基（生命值）
- 可以被施策指定
- 可以发生易席

### 7.2 施策

施策是即时生效的咒语牌，类似于传统卡牌游戏中的"法术"。

**特点**：
- 立即生效
- 执行后进入弃牌堆
- 可以指定立论为目标
- 部分有持续效果

### 7.3 事件

事件是特殊的一次性效果牌。

**特点**：
- 触发后执行特定效果
- 部分在特定条件下触发

### 7.4 场地

场地是持续生效的全局效果牌。

**特点**：
- 持续整局
- 影响所有单位
- 通常有强力但有代价的效果

### 7.5 技能

技能是角色特有的特殊能力。

**特点**：
- 通常与角色绑定
- 有冷却时间
- 可以主动或被动触发

## 八、重要概念解释

### 8.1 辩锋

辩锋相当于卡牌的"攻击力"，决定立论在争鸣中的胜负。

### 8.2 根基

根基相当于卡牌的"生命值"，归零时立论离场。

### 8.3 用度

用度相当于"法力费用"，用于支付打出卡牌的代价。

### 8.4 大势

大势是游戏的核心资源，率先达到8点即获胜。

### 8.5 筹

旁议胜可获得1个筹，最多持有1个。消耗筹可令本回合出牌-1用度。

### 8.6 护印

护印可以吸收伤害，保护立论或主辩者。

### 8.7 易席

易席是指立论在主议和旁议之间移动。
