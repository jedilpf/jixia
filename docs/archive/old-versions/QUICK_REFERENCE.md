## ⚖️ 平衡性警戒线

## 🎓 设计原则速查

### ✅ 好的设计

- [x] 玩家一眼能懂为什么分左中右
- [x] 每路都有独特玩法和胜利方式
- [x] 无绝对最优解，存在克制关系
- [x] 操作技巧能影响胜负
- [x] 资源管理重要但不枯燥

### ❌ 坏的设计

- [ ] 三路规则完全不同（学习成本高）
- [ ] 某路明显强于其他路（不平衡）
- [ ] 存在 T0 卡牌（环境单一）
- [ ] 先手优势过大（>55%）
- [ ] 只有一套主流卡组（缺乏多样性）
