# 核心信息提取（2026-03-09 基线）

## 1. 产品一句话

《百家争鸣》是围绕论场匹配展开、以明辩/暗谋同步博弈为节奏、以三席双层布阵与着书成长为核心的卡牌对战游戏。

## 2. 对局骨架

- 目标时长：约 3~4 分钟，约 6 回合
- 每回合：明辩 10 秒 -> 暗谋 10 秒 -> 揭示 -> 五层结算
- 匹配前必须选择论场，同论场才进入同池

## 3. 战场规则

- 三席：先声席 / 主辩席 / 余论席
- 每席双层：前席 / 后席
- 默认承压传递：前席 -> 后席 -> 主辩者

## 4. 资源系统

- 心证：主辩者生命，归零即败
- 灵势：本回合费用
- 护印：优先承伤
- 证立：短时强化
- 失序：负面节奏
- 文脉：着书带来的长期成长

## 5. 五大牌类

- 立论：正面推进
- 策术：调度修正
- 反诘：应对反制
- 门客：持续站场
- 玄章：持续规则体

## 6. 首发四论场

- 稷下学宫：标准环境
- 火德论坛：前线压制环境
- 藏书秘阁：着书成长环境
- 玄机观星台：信息差暗谋环境

## 7. 160 张设定集使用原则

- 卡牌能力必须服务明辩/暗谋博弈，不做纯数值填充
- 卡牌价值必须体现席位经营，不做无位置感设计
- 着书相关卡必须给到中后期兑现感
- 论场差异要改变打法倾向，但不能重做底层规则
