# 「开始辩斗」简化版策划案

## 一、核心规则（精简版）

| 规则项 | 说明 |
|--------|------|
| **胜利条件** | 把对方血量打空（血量归零即获胜） |
| **议区数量** | 2个：主议 + 旁议 |
| **议区容量** | 每个议区最多放 3 张牌 |
| **卡牌类型** | 只有 2 种：立论（单位）、施策（法术） |
| **出牌限制** | 不限数量，只要有足够费用 |
| **回合流程** | 明辩 → 暗谋 → 揭示 → 结算 |
| **费用曲线** | R1=2, R2=3, R3=4, R4+=5 |

---

## 二、资源系统（只有4个）

| 资源 | 说明 | 代码字段 |
|------|------|----------|
| **费用** | 出牌消耗，每回合刷新 | `cost` |
| **血量** | 玩家生命，归零即输 | `hp` |
| **辩锋** | 单位攻击力（卡牌属性） | `power` |
| **根基** | 单位生命值（卡牌属性） | `hp` |

**删除所有其他资源**：大势、筹、护印、心证、灵势、证立等全部删掉。

### 4.1 Resources（精简版）

```typescript
export interface Resources {
  cost: number;       // 当前费用

}
```

### 4.2 DebateCard

```typescript
export interface DebateCard {
  id: string;
  name: string;
  type: '立论' | '施策';   // 只有两种
  cost: number;            // 费用
  power: number;           // 辩锋（立论牌才有）
  hp: number;              // 根基（立论牌才有）
  effectKind?: 'damage' | 'draw' | 'heal';  // 施策牌效果类型
  effectValue?: number;    // 效果数值

```

### 4.3 SeatState

```typescript
export interface SeatUnit {
  id: string;
  cardId: string;
  name: string;
  power: number;   // 辩锋

export interface SeatState {
```

## 五、结算逻辑（核心）

### 5.1 单位互打

```typescript
function resolveCombat(state: DebateBattleState): void {
  // 主议区：双方单位互打
  const playerZhuYi = state.player.seats.zhu_yi;
  const enemyZhuYi = state.enemy.seats.zhu_yi;
  
  // 我方总辩锋 → 打敌方血量
  const playerPower = playerZhuYi.units.reduce((sum, u) => sum + u.power, 0);
  state.enemy.resources.hp -= playerPower;
  
  // 敌方总辩锋 → 打我方血量
  const enemyPower = enemyZhuYi.units.reduce((sum, u) => sum + u.power, 0);
  state.player.resources.hp -= enemyPower;
  
  // 单位互打（根基扣除）
  for (const playerUnit of playerZhuYi.units) {
    // 找一个敌方单位对打
    const target = enemyZhuYi.units[0];
    if (target) {
      target.hp -= playerUnit.power;
      playerUnit.hp -= target.power;
    }
  }
  
  // 旁议区同理...
}
```

### 5.2 胜利判定

```typescript
function checkVictory(state: DebateBattleState): 'player' | 'enemy' | null {
  // 血量归零即获胜
  if (state.player.resources.hp <= 0) return 'enemy';
  if (state.enemy.resources.hp <= 0) return 'player';
  return null;
}
```

## 八、AI策略（最简版）

```typescript
function aiSimplePlan(state: DebateBattleState): void {
  const enemy = state.enemy;
  const budget = enemy.resources.cost;
  
  // 随机选能打的牌放到议区
  const playable = enemy.hand.filter(card => card.cost <= budget);
  
  // 优先放立论牌（单位）
  const lilunCards = playable.filter(c => c.type === '立论');
  if (lilunCards.length > 0) {
    const card = lilunCards[Math.floor(Math.random() * lilunCards.length)];
    enemy.plan.mainCardId = card.id;
    enemy.plan.mainTargetSeat = 'zhu_yi';  // 默认放主议
    enemy.resources.cost -= card.cost;
  }
  
  // 锁定
  enemy.plan.lockedPublic = true;
}
```
