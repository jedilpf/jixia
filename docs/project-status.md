## 🔴 当前可运行入口

```
npm run dev → http://localhost:5173

流程：
1. MainMenu.tsx 主菜单
   ↓ 点"开始游戏"
2. PreBattleFlow.tsx 预战（选论题、势力、卡牌）
   ↓ 点"开始对战"
3. BattleFrameV2.tsx 侠客论剑战斗页
   - 拖牌到下方规划区
   - 点"规划主策"锁定主卡
   - 点"规划秘策"锁定秘卡
   - 对手AI自动规划（1-2秒延迟）
   - 自动流转到翻示→结算→完成
   - 显示胜负
```

**当前可点击的交互**：
- 菜单按钮 ✅
- 拖拽手牌 ✅
- 规划卡牌 ✅
- 查看卡牌库、日志、状态 ✅
- Pass/Surrender ✅

## 📌 当前最大阻塞

**无明显阻塞** — BattleV2系统已可完整游玩，可演示基本对战。

关键瓶颈（非阻塞，但影响体验）：
1. **AI过于简单** — 对手策略没有深度，影响可玩性
2. **没有持久化** — 无法保存进度，每次都是新局
3. **单机限制** — 不能与其他人联网对战
