# 游戏机制分析：现有原型卡整理

整理日期：2026-03-07

## 1. 核心判断

- 这个项目最该保住的原创点是：`着书`、`简牍区`、`三席二列`。
- 游戏的核心句应当固定为：
  - `这是一个把手牌烧成学识、用前后排站位推进论证路线的卡牌游戏。`
- 现阶段不缺卡牌内容，缺的是：
  - 每张卡在机制中的明确职能
  - 每个门派的清晰赢法
  - 卡牌文本对三席二列站位规则的真实绑定

## 2. 十六派一句话赢法

- 礼心殿：用礼序与前后协同，把每个回合都打成高质量回合。
- 衡戒廷：用规则与禁制，让对手的强回合根本成不了型。
- 归真观：用清明、净化与避伤，把劣势回合熬成优势回合。
- 玄匠盟：用装备、壁垒与工事，把一列打造成无法轻易突破的防线。
- 九阵堂：用起势、破阵与推进，把前排撕开后直接形成终结路线。
- 名相府：用诡辩、负面与错误选择，让对手自己把局面走歪。
- 司天台：用推演、判势与改写未来，让下一轮从开始就偏向自己。
- 游策阁：用合纵连横和跨派联动，把别人的资源变成自己的胜势。
- 万农坊：用积蓄、供养与长线收益，把短期亏损转成后期压制。
- 兼采楼：用兼采百家和工具箱选择，在复杂局面里拼最优解。
- 天工坊：用精密装置、机关联动，形成技术爆发。
- 两仪署：用规则型体系联动、状态转换，构筑阴阳局。
- 杏林馆：用专业治疗、稳态维护与持续净化，打持续战。
- 稗言社：用舆情扰动、群体效应，制造高随机与逼选局。
- 养真院：用个人修行、灵巧生存与自我强化，打灵巧反打。
- 筹天阁：用未来布局、延迟收益，打顶级谋算压制。

## 3. 首发四派建议

- 礼心殿：礼序经营
  - 代表牌：温言立论、竹牍抄录、讲席清规、司史执笔、礼辩同归
- 衡戒廷：规则控制
  - 代表牌：铁券禁令、断简成法、缄印封检、鞫问堂、一纸敕令
- 玄匠盟：工事筑城
  - 代表牌：连弩匣、机关木鸢、城防尺牍、兼济匠师、千机壁垒
- 九阵堂：破阵推进
  - 代表牌：旌旗换阵、战鼓催锋、兵书残卷、锋将·破军、破阵天雷

说明：

- 这四派最容易先跑通清晰的主环境。
- 它们分别代表：秩序、规则、工事、破阵。
- 这四类赢法也最容易绑定三席二列。

## 4. 六个核心状态词

- 护持：常驻防御值，受伤时先扣护持再扣生命。
- 壁垒：临时防御值，默认回合末清空。
- 清明：角色上的领悟层数，适合道家成长结算。
- 气势：玩家资源，适合兵家、阴阳、纵横做中盘爆点。
- 怀疑：玩家负面状态，使其第一张非角色牌更难打出。
- 沉默：角色文本失效，基础身材仍保留。

## 6. 十六门派体系与定位

### 礼心殿（儒家 / ENFJ）
- 核心词：护持、教化、修身、秩序、共鸣
- 定位：中速防守 / 群体增益 / 稳定推进
- 代表气质：温厚、端正、感召、堂正
- 风险：如果不绑定同列前后协同，容易退化成普通抽牌中速

### 衡戒廷（法家 / ESTJ）
- 核心词：禁令、惩戒、封缄、断罪、压制
- 定位：控制 / 封锁 / 规则惩罚
- 代表气质：冷峻、效率、权威、裁断
- 风险：如果做成全局锁死，会强但不高级

### 归真观（道家 / INFP）
- 核心词：清明、净化、闪避、归真、无为
- 定位：续航 / 化解 / 自我稳定
- 代表气质：疏淡、安静、超然、柔韧
- 风险：如果清明只提供续命，会缺少明确胜势

### 玄匠盟（墨家 / ISFJ）
- 核心词：壁垒、节用、兼济、器用、守御
- 定位：装备防守 / 场地保护 / 团队型
- 代表气质：朴实、耐久、可靠、实干
- 风险：如果不围绕某一列做文章，会变成普通堆装备

### 九阵堂（兵家 / ENTJ）
- 核心词：气势、换阵、破军、追击、斩锋
- 定位：爆发 / 斩杀 / 节奏压制
- 代表气质：果断、强势、凌厉、庙算
- 风险：如果只做伤害数值，会变成普通红色进攻派

### 名相府（名家 / ENTP）
- 核心词：反诘、定名、怀疑、同异、诡辩
- 定位：技术反制 / 逻辑拆解 / 干扰
- 代表气质：锋利、机敏、狡黠、智斗
- 风险：和衡戒廷重叠过多

### 司天台（星占家 / INFJ）
- 核心词：预示、星轨、判定、司辰、更迭
- 定位：牌序控制 / 时序操作 / 判定收益
- 代表气质：神秘、深邃、静观、远察
- 风险：随机性过强会稀释“谋算感”

### 游策阁（纵横家 / ESTP）
- 核心词：盟约、游说、借势、离间、互利
- 定位：资源扰动 / 交换 / 非对称节奏
- 代表气质：灵活、世故、精明、应变
- 风险：容易变成万金油

### 万农坊（农家 / ESFJ）
- 核心词：耕养、丰收、屯积、社治、复苏
- 定位：回复 / 长线成长 / 稳健滚雪球
- 代表气质：温暖、厚重、社区性、民生感
- 风险：若只剩回血与护持，会和礼心殿、归真观重叠

### 兼采楼（杂家 / ENFP）
- 核心词：兼采、旁通、综览、博闻、杂取
- 定位：多类型联动 / 复制 / 适配型
- 代表气质：灵动、广博、活跃、好奇
- 风险：要么过泛，要么没有独占价值

### 天工坊（公输家 / ISTP）
- 核心词：机括、构件、校准、拆装、连发
- 定位：精密装置 / 机关连动 / 技术爆发
- 代表气质：冷静、精巧、实操、器物感

### 两仪署（阴阳家 / INTP）
- 核心词：阴阳、五行、相生、相克、转化
- 定位：规则型门派 / 体系联动 / 状态转换
- 代表气质：抽象、理性、宏观、深思

### 杏林馆（医家 / ISTJ）
- 核心词：诊脉、调息、回春、经络、祛毒
- 定位：专业治疗 / 稳态维护 / 持续净化
- 代表气质：耐心、细致、可信、经验主义

### 稗言社（小说家 / ESFP）
- 核心词：传闻、戏说、哄动、众口、渲染
- 定位：舆情扰动 / 群体效应 / 随机趣味
- 代表气质：热闹、鲜活、机灵、表演感

### 养真院（方技家 / ISFP）
- 核心词：导引、轻身、避厄、服食、驻息
- 定位：个人修行 / 灵巧生存 / 自我强化
- 代表气质：自由、灵性、自然、感受型

### 筹天阁（术数家 / INTJ）
- 核心词：筹算、卜筮、宿命、预支、延迟
- 定位：未来布局 / 延迟收益 / 顶级谋算
- 代表气质：冷峻、深谋、长线、结构化

## 7. 后续设计优先级

### 优先补强

- 让更多现有卡直接吃到三席二列关键词：
  - 前排
  - 后排
  - 同列
  - 换位
  - 破列
  - 登堂

### 优先收束

- 先用现有卡跑通首发四派，不急着继续扩门派规模。
- 名相府要和衡戒廷区分。
- 万农坊要深度绑定着书。
- 兼采楼不要过早做成环境核心。

## 8. 最重要的设计原则

以后每张卡都要回答这个问题：

`它是在帮助本门派更快地按自己的方式赢，还是只是一张泛用强牌？`

如果答不出来，这张卡就该重做、后移，或者降为扩展内容。
