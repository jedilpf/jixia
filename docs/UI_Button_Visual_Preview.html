三、按钮类型展示

Type A: 主行动按钮

用途：核心操作（开始游戏、确认选择、结束回合）
特征：渐变背景 + 发光边框 + 悬停放大

Type B: 次要按钮

用途：辅助操作（返回、取消、查看详情）
特征：深色背景 + 细边框 + 悬停高亮

Type C: 危险按钮

用途：破坏性操作（投降、退出、删除）
特征：红色系渐变 + 警告图标

Type D: 图标按钮

用途：快捷操作（设置、邮件、任务）
特征：圆形/方形 + 图标 + 悬停反馈

四、特殊按钮展示

鸣金按钮 (EndTurnButton)

战斗界面核心按钮，带有齿轮旋转动画和脉冲发光效果
状态变化：鸣金（玩家回合）→ 论道中（AI思考）→ 等待（非玩家回合）

五、使用说明

交互状态规范：

• 默认 (Default)：标准样式，无音效
• 悬停 (Hover)：亮度+10%, scale(1.02~1.05), 播放 playHover()
• 按下 (Active)：scale(0.95~0.97), 播放 playClick()
• 禁用 (Disabled)：opacity: 0.45~0.65, cursor: not-allowed

动画时间规范：

• 悬停过渡：200ms ease-out
• 点击反馈：150ms ease-in-out
• 发光脉冲：2000ms ease-in-out infinite
• 齿轮旋转：4000ms linear infinite
