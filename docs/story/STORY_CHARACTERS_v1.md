## 一、角色设计概述

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 基于门派 | 每个角色代表一个门派的核心思想 |
| 立体塑造 | 角色有完整的性格、动机、成长 |
| 关系动态 | 角色关系随玩家选择变化 |
| 历史融合 | 结合真实历史人物和虚构角色 |

### 1.2 角色类型

| 类型 | 数量 | 说明 |
|------|------|------|
| 核心角色 | 3-5 | 贯穿整个剧情的重要NPC |
| 门派代表 | 16 | 每个门派的代表性人物 |
| 配角 | 10+ | 辅助剧情的次要NPC |
| 路人 | 不限 | 场景填充NPC |

## 二、核心角色设计

### 2.1 玩家化身

**代号**：无名学子

**身份**：
- 背景不明的年轻学子
- 通过入学测试进入稷下学宫
- 身份成谜，拥有特殊潜质

**特征**：
- 无门派倾向（初始中立）
- 可被任何门派接纳
- 身世与主线剧情相关

**属性成长**：
```
初始属性：
- 勇气: 30
- 智慧: 30
- 魅力: 30
- 洞察: 30
- 意志力: 30

属性上限：100
```

## 四、关系系统设计

### 4.1 关系类型

| 关系类型 | 范围 | 变化方式 |
|---------|------|---------|
| 好感度 | 0-100 | 正面行为+, 负面行为- |
| 信任度 | 0-100 | 兑现承诺+, 背叛信任- |
| 敌对度 | 0-100 | 冲突行为+, 和解行为- |
| 亏欠值 | 0-100 | 接受帮助+, 偿还帮助- |

### 4.2 关系阈值

| 阈值 | 关系等级 | NPC行为 |
|------|---------|--------|
| 80+ | 挚友/知己 | 主动帮助、分享秘密 |
| 60-79 | 友好 | 积极对话、适度帮助 |
| 40-59 | 中立 | 正常交流 |
| 20-39 | 冷淡 | 敷衍对话、拒绝帮助 |
| 0-19 | 敌对 | 敌意对话、阻挠行动 |

## 五、角色对话模板

### 5.1 好感度对话变化

```markdown
【NPC：颜回（儒家）】
【场景：讲堂广场初遇】

# 好感度 < 30（冷淡）
颜回（淡淡地）："哦，是你。有何贵干？"
*他继续阅读竹简，不再理会*

# 好感度 30-59（中立）
颜回（礼貌地）："这位学子，有何指教？"
*他放下竹简，等待你说话*

# 好感度 60-79（友好）
颜回（微笑）："原来是新来的师弟！幸会幸会。"
*他起身行礼，态度热情*

# 好感度 80+（挚友）
颜回（欣喜）："你可来了！我正有一事想请教。"
*他拉着你到一旁，热情交谈*
```
