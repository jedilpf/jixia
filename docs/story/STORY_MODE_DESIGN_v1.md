## 第一部分：核心剧情框架

### 1.1 故事背景设定

#### 世界观框架

**时代背景**：春秋战国末期，诸子百家于稷下学宫论辩争鸣

**时间跨度**：故事发生在一个名为"稷下城"的虚构都城，历时三年（玩家可探索不同时间段）

**核心设定**：
```
稷下学宫 ─────────────────────────────────────┐
    │                                           │
    ├── 问书院（玩家起始地）                      │
    │      ├── 新入学的学徒                      │
    │      └── 玩家扮演的角色                    │
    │                                           │
    ├── 百家论坛（核心场景）                       │
    │      ├── 墨家 ── 兼爱、非攻、尚贤、节用     │
    │      ├── 儒家 ── 仁义、礼治、教化、正名     │
    │      ├── 法家 ── 法术、威势、秩序、惩罚     │
    │      ├── 道家 ── 无为、道法、自然、逍遥     │
    │      ├── 兵家 ── 战术、阵法、奇袭、调度     │
    │      ├── 纵横家 ── 博弈、连锁、外交、局势   │
    │      ├── 名家 ── 诘辩、反制、逻辑、名实     │
    │      ├── 阴阳家 ── 节律、预判、天象、命运   │
    │      ├── 农家 ── 养成、资源、根基、持久     │
    │      ├── 医家 ── 修复、续战、解毒、增益     │
    │      ├── 乐家 ── 共鸣、强化、混乱、秩序     │
    │      ├── 历数家 ── 时机、爆发、节奏、预知   │
    │      ├── 礼家 ── 稳定、增益、仪式、典范     │
    │      ├── 杂家 ── 融合、适配、折中、包容     │
    │      ├── 小说家 ── 叙事、扰动、传奇、演绎   │
    │      └── 商家 ── 交易、金币、情报、流通     │
    │                                           │
    └── 稷下宫城（权力中心）                       │
           ├── 宫主 ── 神秘的稷下学宫最高领导      │
           └── 大祭司 ── 学术与权力的核心人物      │
                                                  │
天下大势 ◄─────────────────────────────────────┘
```

#### 核心主题

| 主题 | 描述 | 剧情呈现方式 |
|------|------|--------------|
| **思想之争** | 百家思想碰撞，何为天下正道？ | 各派系辩论、理念冲突 |
| **身份认同** | 你是谁？你相信什么？ | 玩家选择塑造角色立场 |
| **权力与道德** | 为达目的是否可以不择手段？ | 关键抉择点体现 |
| **师徒情谊** | 门派归属与个人成长的矛盾 | 角色关系系统 |
| **天下大势** | 个人命运与历史洪流的交织 | 结局走向决定 |

### 1.2 主要角色设定

#### 玩家角色

**代号**：学徒·无名
**真名**：玩家自定义
**背景**：平民出身，被推荐进入稷下学宫
**特殊能力**：可与所有门派建立关系（无门派归属）

**初始属性**：
```yaml
玩家属性:
  名望: 0          # 在学宫中的声望
  智慧: 5          # 思考和辩论文能力
  魅力: 5          # 社交和说服能力
  勇气: 5          # 面对冲突的态度
  洞察: 5          # 看穿他人意图的能力
```

### 1.3 核心冲突与主题

#### 表层冲突

| 冲突类型 | 冲突双方 | 冲突焦点 |
|----------|----------|----------|
| 门派之争 | 各门派 | 论辩胜负、学术地位、资源分配 |
| 路线之争 | 改革派vs保守派 | 学宫发展方向 |
| 继承人之争 | 各派系继承人 | 下一代领导权 |

#### 深层冲突

| 冲突类型 | 描述 | 对玩家影响 |
|----------|------|------------|
| **信仰冲突** | 你相信"人性本善"还是"人性本恶"？ | 决定可结盟的门派 |
| **方法冲突** | 和平论辩 vs 手段不拘 | 决定玩家的行事风格 |
| **立场冲突** | 个人利益 vs 天下大道 | 决定最终结局 |

#### 三大主线

```
主线一：入学试炼
├── 考验内容：各门派基础论辩
├── 关键抉择：选择与哪个门派建立初步联系
└── 结局影响：决定初始师承

主线二：百家之争
├── 核心事件：年度论辩大会
├── 关键抉择：支持哪派/中立/独立
└── 结局影响：决定派系势力消长

主线三：天下正道
├── 核心事件：稷下宫主选举
├── 关键抉择：你的"道"是什么
└── 结局影响：决定故事最终走向
```

#### 节点级分支（参考底特律变人）

每个章节包含多个"关键决策点"(Key Decision Points)，每个决策点：
- 提供2-4个选项
- 选项影响角色关系值
- 重大决策标记为"🔴 关键抉择"

## 第三部分：互动机制设计

### 3.1 选择系统

#### 对话选择类型

| 类型 | 图标 | 描述 | 影响 |
|------|------|------|------|
| **直言** | 💬 | 直接表达立场 | 勇气判定，影响敢于直言的角色好感 |
| **婉言** | 🤝 | 委婉表达看法 | 魅力判定，影响善于社交的角色好感 |
| **反问** | ❓ | 以问代答 | 洞察判定，影响思维缜密的角色好感 |
| **沉默** | 🤫 | 保持沉默 | 智慧判定，影响智者角色好感 |
| **观察** | 👁 | 先观察再行动 | 不做选择，收集更多信息 |

#### 选择影响追踪

```typescript
// 决策影响追踪结构
interface ChoiceImpact {
  // 立即影响
  immediateEffects: {
    statChanges: Record<StatName, number>;
    flagChanges: Record<FlagName, boolean | number>;
  };

  // 延迟影响（用于后续判定）
  delayedEffects: {
    [chapterId: string]: {
      requiredChoice?: string;
      requiredFlags?: Record<FlagName, boolean>;
      requiredStats?: Record<StatName, number>;
      thenNode?: string;
      elseNode?: string;
    };
  };

  // 关系影响
  relationshipChanges: Record<CharacterId, {
    affection?: number;  // 好感
    trust?: number;      // 信任
    rivalry?: number;    // 敌对
  }>;
}
```

### 3.2 关系系统

#### 角色关系值

| 关系类型 | 数值范围 | 描述 |
|----------|----------|------|
| 好感 (Affection) | -100 ~ +100 | 情感上的喜欢/厌恶 |
| 信任 (Trust) | -100 ~ +100 | 是否相信你的言行 |
| 认同 (Rival) | -100 ~ +100 | 是否视你为同道/对手 |

#### 关系状态

| 状态 | 关系值范围 | 角色行为 |
|------|------------|----------|
| **死敌** | ≤-80 | 主动攻击/刁难 |
| **厌恶** | -79 ~ -30 | 拒绝合作/冷言冷语 |
| **冷淡** | -29 ~ +29 | 公事公办 |
| **友善** | +30 ~ +79 | 主动帮助/支持 |
| **挚友** | ≥+80 | 为你辩护/舍身相助 |

#### 门派声望

| 声望等级 | 值范围 | 解锁内容 |
|----------|--------|----------|
| **陌生** | 0~19 | 只能旁听该派辩论 |
| **学徒** | 20~49 | 可参与该派基础活动 |
| **弟子** | 50~79 | 可获得该派秘传 |
| **传人** | 80~99 | 可代表该派发言 |
| **宗师** | 100 | 该派历史将记载你的名字 |

### 3.3 关系影响示例

```
场景：墨家线·公输盘考验

玩家选择"帮助遭受不公待遇的墨家弟子"

立即影响：
├── 勇气 +3
├── 墨家好感 +15
├── 法家好感 -5
└── 触发Flag: "墨家恩人"

└── 若法家好感<-30：法家线解锁"打压墨家"选项
```

### 3.4 QTE快速反应系统

#### QTE类型

| 类型 | 描述 | 判定方式 |

| **直觉QTE** | 凭直觉选择 | 纯运气 |
| **多选QTE** | 在时限内选择多个选项 | 综合判定 |

## 第七部分：技术实现概要

### 7.1 剧情数据结构

```typescript
// 剧情节点类型
interface StoryNode {
  id: string;
  type: 'dialogue' | 'choice' | 'narration' | 'scene' | 'condition' | 'qte' | 'ending';
  speaker?: CharacterId;
  emotion?: CharacterEmotion;
  content: string;
  background?: string;
  music?: string;
  choices?: Choice[];
  effects?: ChoiceImpact;
  nextNode?: string | ConditionNode;
  conditions?: Condition[];
}

// 选择类型
interface Choice {
  id: string;
  text: string;
  icon?: ChoiceIcon;
  effects: ChoiceImpact;
  nextNode: string;
  conditions?: Condition[];
}

// 条件类型
interface Condition {
  type: 'stat' | 'flag' | 'relationship' | 'chapter';
  target: string;
  operator: 'eq' | 'neq' | 'gt' | 'lt' | 'gte' | 'lte';
  value: any;
}

// 影响类型
interface ChoiceImpact {
  stats?: Record<StatName, number>;
  flags?: Record<FlagName, boolean | number>;
  relationships?: Record<CharacterId, RelationshipChange>;
}
```
