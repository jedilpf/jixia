## 一、核心人设

### 1.1 主角（玩家化身）

**代号**：汝（无名学子）

**姓名**：无名（玩家可自定义，或使用默认名"孟舆"）

**身世背景**：

```
孟舆，姓孟名舆，字子舆，齐国临淄人，年十七。

父孟伯达，为临淄城中一小商贾，家道殷实。
母颜氏，出身临淄颜氏旁支，知书达礼。

孟舆幼年时聪慧过人，过目不忘。
六岁能诵《诗》，八岁通《书》，十岁习《礼》。
邻里皆言此子日后必成大器。

然天有不测风云——
孟舆十二岁那年，父亲因商贾纠纷惹上官司，
为护家人周全，散尽家财，郁郁而终。
临终前，将一封泛黄的信函交于孟舆：
「此信，乃父之至交所留。彼言，日后若有难，
可凭此信入稷下学宫求学。吾儿当谨记——
勿负所托，勿忘本心。」

三年守孝期满，孟舆遂携此信，远赴稷下。
彼时，恰逢稷下学宫广招弟子之年。
```

**家族关系**：
- 父：孟伯达（已故，临淄小商贾）
- 母：颜氏（健在，居于临淄）
- 导师：？（信中所指之"至交"，实为学宫某重要人物）

**性格特质**：

#### 1.2.3 已故恩师·孟伯达（主角之父，精神导师）

```yaml
姓名: 孟伯达
字: 兼年
门派: 纵横家（秘密身份：鬼谷子弟子）

年龄: 享年四十三

身份: 临淄商贾（表面），鬼谷门徒（真实）

═══════════════════════════════════════════════════════════
【人物小传】
═══════════════════════════════════════════════════════════

孟伯达，齐国临淄人，出身商贾之家。

然其人虽为商贾，却志不在此。
彼自幼好学，遍访名师，年十八游于稷下。
彼时，鬼谷子尚在学宫，与诸子论辩。

伯达闻鬼谷子之学，大为折服，
遂拜入门下，习纵横之术、捭阖之策。

然鬼谷子告之：
「汝之材，不在纵横，而在观道。
纵横者，以言成事；
观道者，以心通天。
汝若能超越纵横，方为上乘。」

伯达悟之，遂将鬼谷之学与百家融会贯通，
渐成一家之言——以"观"为核心的处世哲学。

彼与荀况亦为挚友：
- 荀况重"礼"，伯达重"观"
- 二人论辩多年，相互砥砺
- 荀况叹曰：「伯达观人观世，洞若观火，吾不及也。」

后伯达因故离开学宫，归家继承商贾之业。
然其心中之志，从未忘却。

彼临终前，致书荀况与鬼谷子：
「吾儿孟舆，性近于吾，若有机缘，望二位前辈照拂。
吾之道，或许能在彼身上延续。」

═══════════════════════════════════════════════════════════

孟伯达之死：

表面原因：商贾纠纷，惹上官司
真实原因：卷入了一场隐秘的学术与政治斗争

- 孟伯达在稷下时，曾与某派学者发生学术冲突
- 此学者后入秦为官，欲借秦国势力打压异己
- 孟伯达被卷入一场"通敌"诬陷
- 为护家人，伯达选择散财消灾，郁郁而终

此学者之身份，将在游戏中逐步揭示——
可能涉及法家、阴阳家等门派。

═══════════════════════════════════════════════════════════

【关键遗物】

1. 推荐信
   - 致荀况的亲笔信
   - 托付孟舆于学宫

2. 玉佩
   - 鬼谷门徒信物
   - 日后可凭此物联络鬼谷子

3. 《观心录》（残卷）
   - 孟伯达毕生所学之记录
   - 部分内容散落各处，集齐后可解锁隐藏剧情

### 2.2 核心冲突

#### 表层冲突：百家学术之争

| 冲突方 | 核心分歧 |
|--------|----------|
| 儒家 vs 法家 | 德治 vs 法治 |
| 墨家 vs 兵家 | 非攻 vs 战争 |
| 道家 vs 纵横家 | 无为 vs 有为 |
| 名家 vs 阴阳家 | 名实 vs 象数 |

#### 中层冲突：学宫存亡之危

```
【危机来源】

1. 政治渗透
   - 各国使节借求学之名刺探情报
   - 秦国势力开始介入学宫

2. 派系对立
   - 各门派争夺学宫控制权
   - 学术争论逐渐演变为意气之争

3. 信仰危机
   - 部分学者质疑"百家争鸣"之意义
   - 有人主张"天下一统"才是正道
```
