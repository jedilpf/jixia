### 7.9 第八章剧情概要

**第八章·终章·多结局系统**
- 节点：n001-n026 + ending（共32个节点）
- 四种结局：
  1. **完美结局**：天子死于乱军，"影"彻底瓦解，六国共治
  2. **普通结局**：秦军大败但天子逃走，天下陷入新的混乱
  3. **悲剧结局**：联军大败，汝被俘处死
  4. **隐藏结局**：天子透露惊天秘密，汝可选择登基或隐居
- 最终选择影响结局走向

### 7.10 多结局设计

| 结局 | 条件 | 描述 |
|------|------|------|
| 完美结局 | 正面战斗胜利 | 天子死，"影"瓦解，六国共治 |
| 普通结局 | 战斗胜利但有漏网 | 天子逃，"影"残存 |
| 悲剧结局 | 战斗失败 | 汝被处死 |
| 隐藏结局 | 发现天子秘密 | 可选择称帝或归隐 |
