# 后续扩展功能池（基于 v2 战斗方式）

## P1（首发后优先）

- 对局回放与关键节点跳转
- 论场专属战报标签（按席位与阶段统计）
- 新手引导 2.0（明辩/暗谋/着书专题）
- 赛季任务与论场挑战

## P2（生态增强）

- 议题池轮换
- 论场周常活动
- 构筑推荐（按论场和门派）
- 实时平衡看板（胜率、回合时长、暗策价值）

## P3（长期）

- 观战模式与复盘评论
- 赛事房间（自定义规则受限版）
- 卡池扩展工具链（策划配置校验 + 自动回归）

## 扩展准入原则

- 不破坏同步回合骨架
- 不削弱三席站位价值
- 不让着书机制失效
- 不引入第二套并行规则书
