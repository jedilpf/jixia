## 一、游戏概述

**稷下学宫**是一款以战国时期诸子百家为背景的辩论卡牌对战游戏。

**核心玩法**：玩家代表不同学派，通过出牌辩论争夺"大势"，先达到8点大势者获胜。

```
主菜单 → 势力选择 → 议题预览 → 门派选择 → 加载界面 → 战斗 → 结算
```

## 三、战斗系统

### 3.1 回合阶段

每个回合按以下顺序执行：

| 阶段 | 名称 | 说明 |
|------|------|------|
| 1 | `round_start` | 回合开始，获得用度，抽牌 |
| 2 | `hidden_submit` | **暗辩提交** - 双方同时暗下出牌 |
| 3 | `submission_lock` | 提交锁定，不可再修改 |
| 4 | `reveal` | 明辩揭示，展示双方卡牌 |
| 5 | `public_effect` | 公开效果结算 |
| 6 | `zone_resolve` | 议区胜负结算，计算大势 |
| 7 | `issue_burst_check` | 议题推进/引爆判定 |

### 3.2 核心参数

```typescript
{
  victoryMomentum: 8,      // 胜利所需大势
  maxRounds: 20,           // 最大回合数
  initialHandSize: 5,      // 初始手牌数
  actionLimitPerTurn: 3,   // 每回合最多出牌数
  maxCardsPerZone: 3,      // 每个议区最多卡牌数
  deckSize: 24,            // 牌组大小
  manaByRound: [2,3,4,5],  // 各回合用度上限
}
```

### 3.3 用度系统

- 每回合开始时，用度（mana）重置为上限
- 用度上限随回合增长：第1回合2点，第2回合3点，以此类推，上限5点
- 出牌需要消耗用度，等于卡牌左上角的费用值

## 五、议区系统

### 5.1 议区布局

```
┌─────────────────────────────────────────────────┐
│                   【议题区】                      │
│              中央辩论议题展示                     │
├──────────────────────┬──────────────────────────┤
│    【对手主议区】    │    【对手旁议区】         │
│    (max 3张)         │    (max 3张)              │
├──────────────────────┼──────────────────────────┤
│    【我方主议区】    │    【我方旁议区】         │
│    (max 3张)         │    (max 3张)              │
├──────────────────────┴──────────────────────────┤
│                  【手牌区】                       │
│              玩家当前手牌 (max 10张)             │
└─────────────────────────────────────────────────┘
```

### 5.2 议区作用

| 议区 | 作用 |
|------|------|
| **主议区** | 正面战场，势力值直接对决，胜者+1大势 |
| **旁议区** | 侧面战场，辅助配合，可触发连锁效果 |

### 5.3 出牌规则

1. **暗辩阶段**：双方同时选择卡牌放入议区，彼此不可见
2. **出牌限制**：
   - 每回合最多出 `3` 张牌
   - 每个议区最多 `3` 张牌
   - 出牌消耗对应用度
3. **卡牌放置**：
   - 选中的卡牌可放入【主议区】或【旁议区】
   - 点击"提交主议"或"提交旁议"按钮确认

## 六、结算规则

### 6.1 势力值计算

```
势力值 = Σ(卡牌公共势力 + ⌊攻击/2⌋) + 连锁加成
```

### 6.2 连锁加成

| 条件 | 加成 |
|------|------|
| 同门派2张以上 | 每多1张 +1 |
| 同议题方向2张以上 | +1 |
| 相邻卡牌共享comboTag | 每对 +1 |

### 6.3 胜负判定

1. 比较【主议区】势力值，高者获胜，+1大势
2. 比较【旁议区】势力值，触发辅助效果
3. 若势力值相等，判定为平局

### 6.4 胜利条件

- **大势获胜**：任一方大势达到 **8** 点
- **回合获胜**：回合数达到 **20** 回合时，大势高者获胜

## 九、AI对战逻辑

### 9.1 AI决策流程

```typescript
// AI出牌决策
function aiDecide(state: GameState): Action[] {
  // 1. 评估手牌价值
  // 2. 考虑当前局势
  // 3. 预测对手出牌
  // 4. 选择最优策略
}
```

### 9.2 AI行为规则

1. **优先级判断**：
   - 低用度高价值牌优先
   - 能触发连锁的组合优先
   - 解场牌在劣势时优先

2. **议区选择**：
   - 核心牌放入主议区
   - 辅助牌放入旁议区
   - 考虑连锁配合

3. **博弈行为**：
   - 分析对手可能的出牌
   - 预留反制手段
   - 控制用度消耗

## 附录：快速开始

### 新手试炼流程

1. **选择"新手试炼"** - 进入教学关卡
2. **观看教程** - 了解基本操作
3. **完成首局对战** - 实践游戏玩法
4. **获得奖励** - 经验、金币

1. **选择"正式战斗"** - 进入匹配
2. **选择门派** - 从4个候选中选择
3. **进行对战** - 运用策略击败对手
4. **结算奖励** - 根据表现获得奖励
