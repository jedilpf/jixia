## 📋 提交步骤

### 第一步：填写申请表

1. 访问中国版权保护中心官网：www.ccopyright.com.cn
2. 注册/登录账号
3. 进入"软件登记" → "软件著作权登记"
4. 在线填写申请表（参考 `02-参考文档/申请表信息.md`）
5. 打印申请表
6. 签字（个人）或盖章（单位）

**关键信息**：
- 软件全称：**谋天下：问道百家**
- 软件简称：**谋天下**
- 英文名称：**ASKING THE TAO**
- 版本号：**V1.0**

### 第二步：准备身份证明

**个人申请**：
- 身份证正反面复印件
- 在复印件上签字

**企业申请**：
- 营业执照副本复印件
- 加盖公章

### 第三步：完善软件说明书

1. 启动项目：`npx vite --host`
2. 截取 6 张界面截图（参考 `02-参考文档/00-材料打包清单.md`）
3. 将截图插入 `软件说明书_增强版.md`
4. 转换为 PDF 格式
5. 添加连续页码（第 X 页 / 共 Y 页）

### 第四步：打印材料

**打印清单**：
1. 软件著作权登记申请表（1 份）
2. 申请人身份证明（1 份）
3. 软件说明书（1 份）
4. source_code_part1.txt（前 60 页）
5. source_code_part2.txt（后 60 页）

**打印要求**：
- 纸张：A4 纸（70g 以上）
- 打印方式：单面打印
- 装订方式：左侧胶装或线装
- **不可使用订书钉**

### 第五步：装订材料

**装订顺序**（从上到下）：
1. 软件著作权登记申请表
2. 申请人身份证明
3. 软件说明书
4. 源代码前 60 页（source_code_part1.txt）
5. 源代码后 60 页（source_code_part2.txt）

**装订要求**：
- 左侧装订
- 使用胶装或线装
- 装订牢固
- 准备 1 式 2 份

### 第六步：提交申请

**提交方式**（三选一）：

**方式一：在线提交（推荐）**
1. 登录中国版权保护中心官网
2. 上传申请表扫描件
3. 上传其他材料扫描件
4. 提交申请
5. 邮寄纸质材料

**方式二：现场提交**
- 地点：北京市西城区天桥南大街 1 号天桥艺术大厦 A 座 302
- 时间：周一至周五 9:00-16:00

**方式三：邮寄提交**
- 收件人：中国版权保护中心
- 地址：北京市西城区天桥南大街 1 号天桥艺术大厦 A 座 302
- 邮编：100050
