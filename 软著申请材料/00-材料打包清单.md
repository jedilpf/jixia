## 五、截图操作指南

### 5.1 启动项目

```bash
# 在项目根目录执行
npx vite --host
```

访问地址：http://localhost:5173

### 5.2 截图步骤

**步骤 1**：打开浏览器开发者工具
- 按 F12 打开开发者工具
- 点击"切换设备工具栏"（Ctrl+Shift+M）
- 设置分辨率为 1920×1080

**步骤 2**：截取主菜单界面
- 访问首页，显示主菜单
- 确保所有 UI 元素加载完成
- 使用截图工具截取完整界面
- 保存为 `assets/screenshots/01-main-menu.png`

**步骤 3**：截取战斗界面
- 点击"开始辩斗"按钮
- 等待游戏加载完成
- 截取完整战斗界面
- 保存为 `assets/screenshots/02-battle-view.png`

**步骤 4**：截取手牌特写
- 将鼠标悬停在手牌上
- 卡牌会放大并置顶
- 截取放大的卡牌
- 保存为 `assets/screenshots/03-hand-cards.png`

**步骤 5**：截取拖拽攻击
- 拖拽随从攻击敌方目标
- 显示攻击指示线
- 暂停或快速截取
- 保存为 `assets/screenshots/04-drag-attack.png`

**步骤 6**：截取游戏结果
- 等待一方英雄生命值降为 0
- 显示胜利/失败界面
- 截取结果界面
- 保存为 `assets/screenshots/06-game-result.png`

### 5.3 截图工具推荐

**Windows 系统**：
- 自带截图工具：Win+Shift+S
- 浏览器开发者工具：Ctrl+Shift+P → 截图
- 第三方工具：Snipaste、FastStone Capture

**截图要求**：
- 完整界面，无浏览器边框
- 清晰可辨，无模糊
- 文件大小：500KB-2MB
- 格式：PNG

## 六、PDF 转换指南

### 6.1 转换方法

**方法一：使用 Word**
1. 打开 Word，导入 Markdown 文件
2. 调整格式和页边距
3. 插入截图
4. 添加页码（页脚居中）
5. 另存为 PDF

**方法二：使用在线工具**
1. 访问 markdown-pdf.com 或类似网站
2. 上传 Markdown 文件
3. 配置页边距和页码
4. 下载 PDF 文件

**方法三：使用 VS Code 插件**
1. 安装"Markdown PDF"插件
2. 右键点击 Markdown 文件
3. 选择"Markdown PDF: Export (pdf)"
4. 配置页码和页眉页脚

### 6.2 PDF 设置要求

| 设置项 | 要求 | 说明 |
|--------|------|------|
| 纸张大小 | A4 | 210×297mm |
| 页边距 | 上下 2.54cm，左右 3.17cm | Word 默认设置 |
| 页码位置 | 页脚居中 | 格式：第 X 页 / 共 Y 页 |
| 页眉 | 软件名称 | 稷下对决卡牌对战系统 V1.0 |
| 图片质量 | 高 | 确保截图清晰 |
