# NEW_CANON_PROPOSAL

## 用途
用于新增或修改 `canon/anchors.yaml` 的唯一流程。

## 提案模板
1. 提案编号: `NCP-YYYYMMDD-XX`
2. 触发原因: 为什么现有锚点无法覆盖
3. 拟新增或修改锚点:
   - id:
   - domain:
   - title:
   - statement:
4. 影响评估:
   - 会影响哪些卡牌
   - 是否触发回滚风险
5. 决策:
   - `approved` / `rejected` / `defer`
6. 审核人:
7. 决策日期:

## 约束
- 未批准提案不得直接写入 `canon/anchors.yaml`。
- 如果被拒绝，想法必须转入 `backlog.md`。
