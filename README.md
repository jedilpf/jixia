# 百家争鸣 2.0（项目主入口）

本仓库当前以你今天确认的设定为唯一基线：**160 张设定集 + 同步对战战斗机制**。

## 基线文档（先读）

1. `docs/战斗机制总说明_v1.md`
2. `docs/首发4个论场完整设定文档_v1.md`
3. `docs/后期开发总策划_基于战斗方式_v1.md`
4. `docs/知识库-01-项目总览.md` ~ `docs/知识库-06-AI技能说明.md`

如果其他旧文档、旧说明、旧代码实现与上述内容冲突，统一按上述文档执行。

## 当前战斗定义（摘要）

- 同步回合：明辩 10 秒 -> 暗谋 10 秒 -> 揭示 -> 五层结算
- 三席双层：先声席 / 主辩席 / 余论席，每席前席 / 后席
- 伤害传递：前席 -> 后席 -> 主辩者
- 五大牌类：立论 / 策术 / 反诘 / 门客 / 玄章
- 核心资源：心证 / 灵势 / 护印 / 证立 / 失序 / 文脉
- 首发论场：稷下学宫 / 火德论坛 / 藏书秘阁 / 玄机观星台

## 快速启动

```bash
npm install
npm run dev
```

常用命令：

```bash
npm run build
npm run preview
npm run typecheck
```

## 目录说明

- `src/`: 当前可运行前端原型
- `docs/`: 策划、架构、开发知识库与规范文档
- `scripts/`: 辅助脚本

## 当前开发共识

- 文档口径优先于旧原型行为。
- 新功能都要服务“明辩/暗谋 + 三席布阵 + 着书成长”。
- 160 张设定集为内容基线，后续扩展在该基线上迭代。
